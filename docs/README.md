# MCP Router Documentation

This folder contains **actionable, version-controlled documentation** for the MCP Router project.

## Structure

```
docs/
├── README.md              # This file
├── ARCHITECTURE.md        # System design and technical decisions
├── API.md                 # API reference (when ready)
├── decisions/             # Architecture Decision Records
│   └── 000-template.md    # ADR template
└── scratch/               # Temporary working files (not for long-term)
```

## What Goes Here vs Obsidian

| Here (docs/) | Obsidian |
|--------------|----------|
| Final technical specs | Brainstorming and ideas |
| API documentation | Research notes |
| Architecture decisions | Requirements gathering |
| Code examples | Daily project logs |
| Deployment guides | Questions and explorations |

## Workflow

1. **Brainstorm** in Obsidian freely
2. **Research** and link ideas in Obsidian
3. **Promote** mature content here when ready to implement
4. **Reference** these docs from code comments

- `/HISTORY.md` - Claude Code session logs
- `/README.md` - Project overview for GitHub
